## 👨‍💻 About Me

</div>

<br/>

I'm **Muhammad Faaez Mugse**, an Android Developer and Computer Science undergraduate at **FAST NUCES Karachi** (Class of 2027). I build modern, production-quality Android applications using **Kotlin and Jetpack Compose**, with a strong focus on clean architecture, smooth user experiences, and real-world impact.

My work spans parental control systems, productivity tools, and accessibility-driven applications — all built with attention to performance, design, and usability. I believe great software is invisible: the user just feels how well it works.

Currently serving as a **Teaching Assistant for Programming Fundamentals (Spring 2026)**, where I help first-year students build strong foundational thinking in software development.

Beyond Android, I have hands-on experience with modern web technologies including **React, Next.js, Node.js, and Firebase** — giving me a full-stack perspective that informs how I design mobile experiences.
